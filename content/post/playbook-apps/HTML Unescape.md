# HTML Unescape

This app unescapes html such as:

- `&lt;p&gt;` => `<p>`
- `mice &amp; men` => `mice & men`
- `&lt;span style=&quot;color: scarlet;&quot;&gt;A&lt;/span&gt;` => `<span style="color: scarlet;">A</span>`
